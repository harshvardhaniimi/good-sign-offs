A bilingual collection of poetic, warm, and witty ways to end your emails.

> 2,100+ English sign-offs from [Meg Miller's Are.na channel](https://www.are.na/meg-miller/good-sign-offs) and 40+ original Hindi sign-offs.

Check out the awesome collection of email sign-offs at [https://signoffs.netlify.app/](https://signoffs.netlify.app/)

## What is this?

A single-page website that serves as both a randomizer and a browsable collection of email sign-offs. Open it, click a button, and get a beautiful way to end your next email.

The English sign-offs are pulled live from the Are.na API — a crowd-sourced collection that's been growing since 2019. The Hindi sign-offs (हिंदी विदाइयाँ) are originals, written to capture the warmth and poetry of Hindi and Urdu.

## Features

- **Randomize** — get a fresh sign-off with one click
- **Browse all** — scroll through the full collection
- **Bilingual display** — English and Hindi shown side by side
- **Copy to clipboard** — one tap to copy either language
- **Download** — export all sign-offs as a Markdown file with credits

## How it's built

- Single HTML file with embedded CSS and JavaScript
- Are.na API for live English sign-offs
- Google Fonts: Playfair Display, Source Serif 4, Mukta
- Hindi translations generated with a custom Python script (`translate.py`)
